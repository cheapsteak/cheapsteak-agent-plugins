# web-tools

Driving a real browser to verify UI behavior.

## Skills

- **browse** — Browse any web app in a real browser using the Playwright CLI. Use when asked to "check the app", "open the site", "look at the UI", "verify this page", or diagnose frontend behavior. General-purpose — not tied to any specific app.

## Requirements

- `npx` / Node.js (Playwright CLI runs via `npx playwright`)
